## Syllabus for Class 3
* HTML concepts
  - HTML doctype
  - HTML entity codes: `&nbsp;` `&lt;` `&gt;` `&quot;`
  - meta tags: charset, description, keywords
  - del
  - ins
  - u (vs `ins`)
  - q (vs `&quot;`)
  - blockquote
  - pre, code
  - page layout using `table` element
  - vertical-align
  - box-sizing: `border-box`
* CSS concepts
  - text-align
  - border-radius
  - type selector
  - class selector
  - id selector
  - wildcard selector (`*`)
